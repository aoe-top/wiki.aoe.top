# 入门清单

新用户使用 Bing Webmaster Tools（必应站长工具）？下面的清单可以帮助你快速上手。第 1 步和第 2 步会让你启动并运行；第 3 步和第 4 步为可选但强烈推荐。

## 第 1 步：创建 Bing Webmaster Tools 帐户

前往 Bing Webmaster Tools 登录/注册 页面并创建一个免费的帐户：

- Bing Webmaster Tools 登录或注册：https://www.bing.com/toolbox/webmaster/

你可以使用现有的 Microsoft、Google 或 Facebook 帐户注册。或者你也可以为 Bing Webmaster Tools 新建一个 Microsoft 帐户：在注册时选择 Microsoft，然后在下一步点击“Create one”创建新帐户。

## 第 2 步：添加并验证你的网站

创建帐户后，你可以将网站添加到你的站长工具中：

- 添加站点说明：https://www.bing.com/webmasters/help/add-and-verify-site-12184f8b

添加站点后，需要验证你对该网站的所有权。请在左侧导航中打开“Add and Verify Site”部分，按照分步说明完成添加与验证。Bing 提供多种所有权验证方法（Ownership Verification methods），请选择最适合你站点属性的方法并按页面说明操作。

你可以添加多个站点，并通过“User Management”（用户管理）功能控制访问权限：

- 用户管理说明：https://www.bing.com/webmasters/help/how-to-add-users-to-your-site-account-d5d00364

如果验证成功，你会看到绿色勾选及域名已成功添加的提示。如果看到红色叉号，请查看错误信息并根据提示解决问题后重试。你也可以删除已添加的站点，但如果再次添加则需要重新验证。

## 第 3 步：创建并上传网站地图（Sitemaps）

通过添加网站地图（Adding Sitemaps）可以向 Bing 告诉你站点上的 URL，Bing 支持多种网站地图格式：

- XML Sitemap 与 XML Sitemap 索引文件（遵循 http://www.sitemaps.org 的规范）
- Atom 0.3 / 1.0
- RSS 2.0
- 文本文件（每行一个 URL）

更多信息与说明请参阅：

- 添加网站地图：https://www.bing.com/webmasters/help/sitemaps-3b5cf6ed
- sitemaps.org：http://www.sitemaps.org

通常网站地图放在站点根目录，例如你的网站为 www.contoso.com，则网站地图通常位于 http://www.contoso.com/sitemap.xml。

网站地图创建后，上传到服务器并通过 Bing Webmaster Tools 提交即可。在站长工具左侧的 Sitemaps 部分可以管理与提交站点地图。

## 第 4 步：制定搜索优化计划（SEO）

使用 Bing Webmaster Tools 中的 SEO 工具可以简化你的搜索引擎优化计划。在站长工具左侧的 SEO 栏中，你会找到用于：

- 分析反向链接（backlinks）
- 关键词研究（keyword research）
- 检查并修复 SEO 错误

Bing 会对你已验证的域名每隔一周左右自动运行一次 SEO 报告（平均每两周一次），并提供改进建议。如果你的域名刚完成验证，可能需要等待几天才能看到首次报告。
