# 为什么我的网站不在索引中？

当 Bing 抓取并索引互联网上的页面时，会优先选择那些更有可能满足搜索者需求的网站。互联网上的网站数量几乎是无限的；并非所有页面都能被收录，因此您网站的某些页面可能未包含在 Bing 索引中。本文将告诉您如何验证某个页面是否已被索引、可能导致页面未被收录的原因，以及提高被收录几率的建议。

在了解原因之前，先确认您的页面确实没有被索引。可以使用 `url:` 搜索操作符对网站的特定 URL 进行查询（在操作符后跟上您的页面地址）。如果查询返回了结果，说明该 URL 已被索引。如下示例所示，该 URL 在索引中并作为结果返回：

![URL Operator Query](https://www.bing.com/webmaster/help/helpimage/9e65e7b5_en-us "URL Operator Query")

如果没有返回结果，则说明该站点或页面未被索引。可能的原因包括以下一种或多种情况：

- **您的网站是新站，我们还未发现并抓取：**
	如果您的网站是新建的，Bing 可能需要时间来发现站点链接并抓取页面。即便您通过 [提交站点地图](https://www.bing.com/webmasters/help/Sitemaps-3b5cf6ed) 或使用 Webmaster Tools（参见 [向 Bing 提交 URL](https://www.bing.com/webmasters/help/URL-Submission-62f2860b)）直接提交页面，我们仍需要抓取并评估页面后才会将其收录。

- **我们在抓取站点时遇到问题：**
	如果我们无法稳定地抓取您的网站，可能会选择不收录。您可以使用 [URL 检查](https://www.bing.com/webmasters/help/URL-Inspection-55a30305) 工具来查看 Bingbot（我们的主要爬虫）是否能够抓取您的主页及其他页面。

- **没有指向您网站的外部链接：**
	新站常见的问题是缺乏来自其他网站的链接。这类链接帮助我们发现您的内容并反映页面的受欢迎程度。与其有大量来自低质量站点的新链接，不如获得少量来自优质权威网站的链接。至少需要有一个外部链接指向您的网站以便我们找到它。通过创建有吸引力且值得链接的内容来争取优质外链。

- **您的 robots.txt 文件阻止我们抓取站点：**
	如果您错误配置了 robots.txt，可能会告知 Bingbot 不去抓取网站，从而导致无法被索引。请参考 [如何创建 Robots.txt 文件](https://www.bing.com/webmasters/help/?topicid=cb7c31ec) 以了解基本设置方法。

- **页面中包含 `<meta name="robots" content="NOINDEX">`：**
	如果页面源码中包含该元标记，我们在抓取时会识别到并不会将其加入索引。

- **您在 Bing Webmaster Tools 中屏蔽了 URL：**
	若您拥有 Bing Webmaster Tools 帐户，可能使用了 [屏蔽 URL](https://www.bing.com/webmasters/help/Block-URLs-264e560b) 工具阻止某些地址出现在结果中。请检查账户设置，确保预期出现在索引中的 URL 未被屏蔽。

- **您的站点未达到 Bing 要求的质量阈值：**
	Bing 更喜欢独特且高质量的内容。请确保您的网站没有大量重复内容、不会快速重定向用户或提供过于浅薄的内容。长期的低质量表现可能导致站点被移出索引，以便为更优质的网站腾出空间。

- **您的网站被处罚并从索引中移除：**
	在极少数情况下，如果我们发现您的网站违反质量指南或存在恶意软件，站点可能被从搜索结果中移除。Bing 会通过 Webmaster Tools 直接与网站所有者沟通并在检测到恶意软件时发出警告。如果检测到恶意软件，我们会从搜索结果中完全移除站点或在其条目上添加恶意软件警告以保护用户。如果您的站点受到恶意软件影响，请参阅 [Bing Webmaster Guidelines](https://www.bing.com/webmasters/help/Webmaster-Guidelines-30fba23a)。
