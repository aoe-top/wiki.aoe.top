# 为什么我会不断收到来自 Webmaster Tools 的电子邮件通知？

如果您正在收到来自 Bing Webmaster Tools 的问题通知邮件，这通常表示您的 [通知中心](https://www.bing.com/webmasters/help/?topicid=2d8ad7ca) 中有**未读消息**，并且您已在 [电子邮件首选项](https://www.bing.com/webmasters/help/?topicid=c73a478c) 中启用了将通知发送到邮箱的设置。系统会将**新的和未读的消息**通过邮件提醒您。

要更改邮件设置，请登录 Bing Webmaster Tools，然后点击页面顶部的“通知”链接更改首选项。如果您仍希望接收将来的问题通知，请在 [通知面板](https://www.bing.com/webmaster/help/?topicid=2d8ad7ca) 中将以前的未读消息标记为已读、存档或删除，这样以后只会收到新的消息通知。
