# HTTPS：Bingbot 支持哪些密码套件？

如果 Bingbot 抓取您启用 HTTPS 的网站时遇到问题，请确保您的站点至少配置了下面列表中的一种密码套件。注意，自 2015 年秋季起，Bingbot 已停止抓取仅支持 [RC4](https://en.wikipedia.org/wiki/RC4) 的站点，因为 RC4 存在多种漏洞且被视为不安全。大多数浏览器（包括 Chrome、Firefox 及 Microsoft Edge）自 2016 年初已停止支持 RC4。
