# 如何提交关于 Bing Webmaster Tools 的反馈？

我们欢迎来自站长/网站所有者的反馈，致力于不断提升您使用 Bing Webmaster Tools 的体验。提交反馈非常简单：在 Webmaster Tools 门户右上角找到 **笑脸（Smiley）** 图标并点击即可。

![Feedback](https://www.bing.com/webmaster/help/helpimage/feedback "Feedback_NPS")

> **注意：**

> Bing 无法对单个反馈进行逐一回复，且该渠道并非用于支持性询问。但 Bing 可能会使用汇总的反馈数据来帮助确定对 Bing Webmaster Tools 服务的改进优先级。请阅读我们的 [隐私声明](http://go.microsoft.com/fwlink/?LinkId=248686) 和 [服务协议](https://www.microsoft.com/en-us/servicesagreement)。
